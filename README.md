# 🚀 Apex Optimizer

**Otimize seu PC e maximize o FPS no Apex Legends**

Conjunto de Ferramentas simples para aplicar configurações otimizadas no Windows e no Apex Legends.

---

## 🎯 Objetivo

Este projeto foi criado para quem tem sistemas com desempenho limitado ou sofrem com algum tipo de Input Lag, vice versa.

---

## ⚙️ Funcionalidades

- 🔧 Ajustes no Windows para reduzir uso desnecessário de recursos  
- 🎮 Otimizações específicas para Apex Legends  
- ⚡ Melhoria de FPS e estabilidade  
- 📉 Redução de input lag  
- 🧠 Configurações voltadas para gameplay competitivo  

---

## 🚀 Como usar

1. Baixe os arquivos do repositório  
2. Execute como **Administrador**  
3. Siga as instruções exibidas na tela  
4. Reinicie o sistema após terminar de aplicar os Comandos e Arquivos (**recomendado**)  

---

## 📦 Requisitos

- Windows 10 ou 11  
- Permissão de Administrador  
- Apex Legends instalado  (Steam ou EA)

---

## ⚠️ Aviso importante

- As otimizações modificam configurações do sistema  
- Recomenda-se criar um **ponto de restauração** antes de usar  
- Use por sua conta e risco
- Cuidado selecionando o Buffer Errado (Otimização de Ram)
